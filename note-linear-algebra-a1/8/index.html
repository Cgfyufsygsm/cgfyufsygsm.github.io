矩阵的逆

设 A∈Mm×n(K)A\in M_{m\times n}(K)A∈Mm×n​(K)，若 ∃B\exists B∃B 使得 AB=IAB=IAB=I，则称 BBB 为 AAA 的右逆；若 ∃B\exists B∃B 使得 BA=IBA=IBA=I，则称 BBB 为 AAA 的左逆。

何时有右逆？当且仅当 AAA 的列能表出 KmK^mKm 的标准基 ε1,⋯ ,εm\varepsilon_1, \cdots ,\varepsilon_mε1​,⋯,εm​，故 AAA 行满秩，即 rank⁡(A)=m\operatorname{rank}(A) = mrank(A)=m。同理，矩阵有左逆当且仅当 AAA 列满秩，rank⁡(A)=m\operatorname{rank}(A)=mrank(A)=m。注意：左逆和右逆可能是不唯一的。

若方阵 AAA 的左/右逆均存在且相等，则将其记作 A−1A^{-1}A−1，称 AAA 可逆。下面证明逆的唯一性。

假设 A,B∈Mn(K)A,B\in M_n(K)A,B∈Mn​(K)，满足 AB=IAB=IAB=I，下面证明 BA=IBA=IBA=I。ABA=AABA=AABA=A，A(BA−I)=0A(BA-I)=0A(BA−I)=0。由于 AAA 满秩，故 BA−I=0BA-I=0BA−I=0，即 BA=IBA=IBA=I。

番外：投影矩阵

现在考虑对于列满秩矩阵 AAA，构造其的一个左逆。

由于 rank⁡(ATA)=rank⁡(A)\operatorname{rank}(A^TA)=\operatorname{rank}(A)rank(ATA)=rank(A)，故 ATAA^TAATA 满秩，可逆。又由 (ATA)−1ATA=I(A^TA)^{-1}A^TA = I(ATA)−1ATA=I，故 (ATA)−1AT(A^TA)^{-1}A^T(ATA)−1AT 为 AAA 的一个左逆，令其为 A+A^+A+。

这个东西的用处在于其可以将 β\betaβ 投到 C(A)\mathcal{C}(A)C(A) 中。比如对于一个无解的方程组 AX=βAX=\betaAX=β，我们要求他的一个近似解，自然我们需要找到一个“最近”的 γ\gammaγ 使得 γ∈C(A)\gamma\in \mathcal{C}(A)γ∈C(A)，然后解 AX=γAX=\gammaAX=γ。而 AA+A A^+AA+ 就是一个很好的投影矩阵。AA+βA A^+\betaAA+β 即为 β\betaβ 在 C(A)\mathcal{C}(A)C(A) 上的正交投影。而取 X=A+βX=A^+\betaX=A+β 作为解亦是自然的。

下面证明为何其为正交投影：AA+β∈C(A)A A^+\beta\in \mathcal{C}(A)AA+β∈C(A) 是自然的，然后我们用 AAA 中的列向量与 β−γ\beta - \gammaβ−γ 做内积，这相当于用 ATA^TAT 乘上他，就是 AT(β−AA+β)=AT(I−A(ATA)−1AT)β=AT(I−AA−1(AT)−1AT)β=0A^T(\beta - A A^+\beta)=A^T(I - A(A^TA)^{-1}A^T)\beta=A^T(I - A A^{-1}(A^T)^{-1}A^T)\beta = 0AT(β−AA+β)=AT(I−A(ATA)−1AT)β=AT(I−AA−1(AT)−1AT)β=0，证毕。

类似的，AAA 行满秩时，A+=AT(AAT)−1A^+ = A^T(AA^T)^{-1}A+=AT(AAT)−1 为其最好的一个右逆。AA+=IA A^+=IAA+=I，A+AA^+AA+A 是向 AAA 行空间做投影的矩阵。

一般而言，计算矩阵的逆不使用伴随矩阵法，计算量太大。我们基于任何满秩方阵都能分解为 P1⋯PsP_1\cdots P_sP1​⋯Ps​ 的事实，我们若能找到一系列初等矩阵满足 P1P2⋯PsA=IP_1P_2\cdots P_sA = IP1​P2​⋯Ps​A=I，那么是不是就有 A−1=P1P2⋯PsA^{-1} = P_1P_2\cdots P_sA−1=P1​P2​⋯Ps​。

定理：对于 A∈Mn(K)A\in M_n(K)A∈Mn​(K)，以下五个条件等价：

  1. AAA 可逆；
  2. ∃B\exists B∃B s. t. AB=IAB=IAB=I（或 BA=IBA=IBA=I）；
  3. rank⁡(A)=n\operatorname{rank}(A) = nrank(A)=n；
