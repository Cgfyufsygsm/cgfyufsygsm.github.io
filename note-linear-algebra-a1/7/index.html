矩阵乘积的秩

定理：设 As×nA_{s\times n}As×n​，Bn×mB_{n\times m}Bn×m​，则

  1. rank⁡(AB)≤min⁡{rank⁡(A),rank⁡(B)}\operatorname{rank}(AB)\le \min\left\{ \operatorname{rank}(A),\operatorname{rank}(B) \right\}rank(AB)≤min{rank(A),rank(B)}

    证明应该是容易的，按照一开始提到的，ABABAB 的列是由 AAA 的列表出的，故 ABABAB 的列秩 ≤rank⁡(A)\le \operatorname{rank}(A)≤rank(A)；同理，ABABAB 的行是由 BBB 的行表出的，所以 ABABAB 的行秩 ≤rank⁡(B)\le \operatorname{rank}(B)≤rank(B)。

  2. 若 AB=0AB=0AB=0，则 rank⁡(A)+rank⁡(B)≤n\operatorname{rank}(A)+\operatorname{rank}(B)\le nrank(A)+rank(B)≤n

  3. rank⁡(A)+rank⁡(B)≤n+rank⁡(AB)\operatorname{rank}(A)+\operatorname{rank}(B)\le n + \operatorname{rank}(AB)rank(A)+rank(B)≤n+rank(AB)
    来一起证明 222 和 333。记 B=[β1⋯βm]B = \begin{bmatrix} \beta_1 & \cdots & \beta_m \\\end{bmatrix}B=[β1​​⋯​βm​​]，则 AB=[Aβ1⋯Aβm]AB = \begin{bmatrix} A\beta_1 & \cdots & A\beta_m \\\end{bmatrix}AB=[Aβ1​​⋯​Aβm​​]，设 rank⁡(AB)=r\operatorname{rank}(AB)=rrank(AB)=r，WLOG 设 Aβ1,⋯ ,AβrA\beta_1, \cdots ,A\beta_rAβ1​,⋯,Aβr​ 为 ABABAB 的列极大无关组，于是 Aβr+1,⋯ ,AβsA\beta_{r+1},\cdots,A\beta_sAβr+1​,⋯,Aβs​ 可以由 Aβ1,⋯ ,AβrA\beta_1, \cdots ,A\beta_rAβ1​,⋯,Aβr​ 线性表出：

    Aβr+i=ci1Aβ1+ci2Aβ2+⋯+cirAβr1≤i≤s−rA\beta_{r+i} = c_{i1}A\beta_1 + c_{i2}A\beta_2 + \cdots + c_{ir}A\beta_r\quad 1\le i\le s - r Aβr+i​=ci1​Aβ1​+ci2​Aβ2​+⋯+cir​Aβr​1≤i≤s−r

    rank⁡(B)=rank⁡{β1,⋯ ,βr,⋯ ,βr+1,⋯ ,βs}=rank⁡{β1,⋯ ,βr,⋯βr+1′,⋯ ,βs′}≤rank⁡{β1,⋯ ,βr}+rank⁡{βr+1′,⋯ ,βs′}≤r+n−rank⁡(A)\begin{aligned} \operatorname{rank}(B) &= \operatorname{rank}\left\{ \beta_1, \cdots ,\beta_r, \cdots ,\beta_{r+1}, \cdots ,\beta_s \right\}\\ &= \operatorname{rank}\left\{ \beta_1, \cdots ,\beta_r, \cdots \beta_{r+1}', \cdots ,\beta_s' \right\} \\ &\le \operatorname{rank}\left\{ \beta_1, \cdots ,\beta_r \right\} + \operatorname{rank}\left\{ \beta_{r+1}', \cdots ,\beta_s' \right\}\\ &\le r + n - \operatorname{rank}(A) \end{aligned} rank(B)​=rank{β1​,⋯,βr​,⋯,βr+1​,⋯,βs​}=rank{β1​,⋯,βr​,⋯βr+1′​,⋯,βs′​}≤rank{β1​,⋯,βr​}+rank{βr+1′​,⋯,βs′​}≤r+n−rank(A)​

    证毕。

    注意在这个证明过程中，满足条件 Aη=0A\eta = \boldsymbol{0}Aη=0 的向量 η\etaη 在 AX=0AX=\boldsymbol{0}AX=0 的解空间内是很重要的思路。很多时候利用这个性质可以列 dim⁡W≤n−rank⁡(A)\operatorname{dim} W\le n - \operatorname{rank}(A)dimW≤n−rank(A) 来解决问题。

  4. 根据 3 给出的重要不等式，可以得出如下推论：

    • 左乘列满秩的矩阵，矩阵的秩不变；
    • 右乘行满秩的矩阵，矩阵的秩不变；
    • 左乘/右乘满秩矩阵，矩阵的秩不变。

实矩阵的一个重要性质

在继续之前，我们先引出一条重要的引理：若实向量 α\alphaα 满足 αTα=0\alpha^T\alpha = 0αTα=0，则 α=0\alpha = 0α=0。证明：αTα=α⋅α=a12+a22+⋯+an2≥0\alpha^T\alpha= \alpha\cdot \alpha = a_1^2 + a_2^{2} + \cdots + a_n^{2}\ge 0αTα=α⋅α=a12​+a22​+⋯+an2​≥0，等号成立当且仅当 α=0\alpha = 0α=0。

定理：若 A∈Mm×n(R)A\in M_{m\times n}(\mathbb{R})A∈Mm×n​(R)，则

  1. N(ATA)=N(A)\mathcal{N}(A^TA)= \mathcal{N}(A)N(ATA)=N(A)，其中 N(A)\mathcal{N}(A)N(A) 表示齐次线性方程组 AX=0AX=\boldsymbol{0}AX=0 的解空间。

    考虑 ∀η∈N(ATA)\forall \eta\in \mathcal{N}(A^TA)∀η∈N(ATA)，有 ATAη=0A^TA\eta = \boldsymbol{0}ATAη=0，左乘一个 ηT\eta^TηT，ηTATAη=0\eta^TA^TA\eta = \boldsymbol{0}ηTATAη=0，这即 (Aη)TAη=0(A\eta)^TA\eta = \boldsymbol{0}(Aη)TAη=0，根据引理，Aη=0A\eta = \boldsymbol{0}Aη=0，所以 N(ATA)⊆N(A)\mathcal{N}(A^TA) \subseteq \mathcal{N}(A)N(ATA)⊆N(A)，证毕。

矩阵乘积的行列式

我们通过 PJ 分解相关理论，可以知道满秩方阵都能分解成初等矩阵的乘积。（满秩阵的 rref 为单位阵）

随后我们证明一个引理：对于初等矩阵 PPP 有 ∣PB∣=∣P∣∣B∣|PB|=|P||B|∣PB∣=∣P∣∣B∣。分别考虑三种初等变换对行列式的影响即可，这里不写具体证明了。

定理：方阵积的行列式等于方阵行列式的积，即 ∣AB∣=∣A∣∣B∣|AB|=|A||B|∣AB∣=∣A∣∣B∣。

证明：若 AAA 不满秩，则 ∣A∣=0|A|=0∣A∣=0 且 rank⁡(AB)≤rank⁡(A)<n\operatorname{rank}(AB)\le \operatorname{rank}(A)<nrank(AB)≤rank(A)<n，∣AB∣=0|AB|=0∣AB∣=0，成立。

若 AAA 满秩。则 AAA 一定能拆成 P1P2⋯PsP_1P_2\cdots P_sP1​P2​⋯Ps​ 的形式，所以 ∣AB∣=∣P1P2⋯PsB∣=∣P1∣∣P2⋯PsB∣=∣P1∣∣P2∣⋯∣Ps∣∣B∣|AB| = |P_1P_2\cdots P_s B| = |P_1||P_2\cdots P_sB| = |P_1||P_2|\cdots|P_s||B|∣AB∣=∣P1​P2​⋯Ps​B∣=∣P1​∣∣P2​⋯Ps​B∣=∣P1​∣∣P2​∣⋯∣Ps​∣∣B∣，由 ∣A∣=∣P1∣⋯∣Ps∣|A| = |P_1|\cdots|P_s|∣A∣=∣P1​∣⋯∣Ps​∣，证毕。
