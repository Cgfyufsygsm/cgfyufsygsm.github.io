映射

具体的概念略，比如从 SSS 到 TTT 的映射 f:S→Tf:S\to Tf:S→T，α\alphaα 映射到 β\betaβ 的话就是 α↦β\alpha \mapsto \betaα↦β。

SSS 是定义域，TTT 是陪域。β\betaβ 称为 α\alphaα 的像，α\alphaα 被称为 β\betaβ 的一个原像。fff 的像集记作 Im⁡f\operatorname{Im} fImf 或 f(S)f(S)f(S)，又称值域。

注意值域与陪域的区别。Im⁡f⊆T\operatorname{Im}f \subseteq TImf⊆T。

  • 单射：∀x1,x2∈S\forall x_1,x_2\in S∀x1​,x2​∈S，若 x1≠x2x_1\neq x_2x1​=x2​ 则一定有 f(x1)≠f(x2)f(x_1)\neq f(x_2)f(x1​)=f(x2​)。
  • 满射：f(S)=Tf(S)=Tf(S)=T。即陪域中每个元素有至少一个原像。
  • 双射/一一对应：陪域中每一个元素有唯一的一个原像。相当于即单又满。

每个元素都映射到自身的变换叫恒同变换，记作 1S1_S1S​。1Tf=f1S1_Tf = f 1_S1T​f=f1S​。

映射的乘法/合成：f:S′→S′′f:S'\to S''f:S′→S′′，g:S→S′g:S\to S'g:S→S′，（要求内层映射的陪域为外层映射的定义域），定义复合 fgfgfg。(fg)(a)=f(g(a))(fg)(a)=f(g(a))(fg)(a)=f(g(a))。

映射复合满足结合律：f:S′′→S′′′f:S''\to S'''f:S′′→S′′′，g:S′→S′′g:S'\to S''g:S′→S′′，h:S→S′h:S\to S'h:S→S′，则 f(gh)=(fg)hf(gh)=(fg)hf(gh)=(fg)h

  • fff 有右逆等价于 fff 为满射。即 f:S→Tf:S\to Tf:S→T 为满射   ⟺  ∃g:T→S\iff \exists g:T\to S⟺∃g:T→S s. t. fg=1Tfg = 1_Tfg=1T​。（严格叙述需要选择公理）
  • fff 有左逆等价于 fff 为单射。即 f:S→Tf:S\to Tf:S→T 为单射   ⟺  ∃g:T→S\iff\exists g:T\to S⟺∃g:T→S s. t. gf=1Sgf = 1_Sgf=1S​。

若 fff 有左逆和右逆，则称 fff 可逆，左逆和右逆唯一且相同。fff 可逆等价于 fff 为双射。

证明：必要性可以考虑证明 fff 为单射且 fff 为满射；充分性就直接从双射的定义出发。

必要性。设 f:S→Tf: S\to Tf:S→T 可逆，则 ∃f−1:T→S\exists f^{-1}:T\to S∃f−1:T→S，且 ff−1=1Tff^{-1} = 1_Tff−1=1T​，f−1f=1Sf^{-1}f=1_Sf−1f=1S​。∀a′∈T\forall a'\in T∀a′∈T，有 f−1(a′)∈Sf^{-1}(a')\in Sf−1(a′)∈S，且 f(f−1(a′))=(ff−1)(a′)=a′=1T(a′)=a′f(f^{-1}(a')) = (ff^{-1})(a')=a'=1_T(a')=a'f(f−1(a′))=(ff−1)(a′)=a′=1T​(a′)=a′，故 a′a'a′ 在 fff 下总有一个原像 f−1(a′)f^{-1}(a')f−1(a′)，故 fff 为满射。

∀a1,a2∈S\forall a_1,a_2\in S∀a1​,a2​∈S，若 f(a1)=f(a2)f(a_1)=f(a_2)f(a1​)=f(a2​)，则 f−1(f(a1))=f−1(f(a2))f^{-1}(f(a_1))=f^{-1}(f(a_2))f−1(f(a1​))=f−1(f(a2​))，即 1S(a1)=1S(a2)1_S(a_1)=1_S(a_2)1S​(a1​)=1S​(a2​)，故 a1=a2a_1=a_2a1​=a2​，故 fff 为单射。从而 fff 为双射。

充分性。由 fff 为双射，∀a′∈T\forall a'\in T∀a′∈T，其在 SSS 下有唯一原像 aaa，此时 f(a)=a′f(a)=a'f(a)=a′。令

g:T→Sa′↦a\begin{aligned} g:T&\to S \\ a'&\mapsto a \end{aligned} g:Ta′​→S↦a​

则 (fg)(a′)=f(g(a′))=f(a)=a′(fg)(a') = f(g(a')) = f(a) = a'(fg)(a′)=f(g(a′))=f(a)=a′，即 fg=1Tfg = 1_Tfg=1T​。且 ∀a∈S\forall a\in S∀a∈S，g(f(a))=ag(f(a))=ag(f(a))=a，即 gf=1Sgf = 1_Sgf=1S​。故 fff 可逆。

线性映射

设 A:Kn→Km\mathcal{A}: K^n\to K^mA:Kn→Km，若 A\mathcal{A}A 满足

  • ∀α,β∈Kn\forall \alpha,\beta \in K^n∀α,β∈Kn，有 A(α+β)=A(α)+A(β)\mathcal{A}(\alpha+\beta)=\mathcal{A}(\alpha)+\mathcal{A}(\beta)A(α+β)=A(α)+A(β)
  • ∀α∈Kn,k∈K\forall \alpha\in K^n,k\in K∀α∈Kn,k∈K，有 A(ka)=kA(a)\mathcal{A}(ka)=k \mathcal{A}(a)A(ka)=kA(a)

则称 A\mathcal{A}A 为线性映射。m=nm=nm=n 时称为 KnK^nKn 上的线性变换。

线性映射可以保持向量组的线性相关性。例如：

  • A(0)=0\mathcal{A}(\boldsymbol{0}) = \boldsymbol{0}A(0)=0
  • A(k1α1+⋯+ksαs)=k1A(α1)+⋯+ksA(αs)\mathcal{A}(k_1\alpha_1+ \cdots +k_s\alpha_s) = k_1 \mathcal{A}(\alpha_1) + \cdots + k_s \mathcal{A}(\alpha_s)A(k1​α1​+⋯+ks​αs​)=k1​A(α1​)+⋯+ks​A(αs​)。
  • 若 α1,⋯ ,αs\alpha_1, \cdots ,\alpha_sα1​,⋯,αs​ 线性相关，则 A(α1),⋯ ,A(αs)\mathcal{A}(\alpha_1), \cdots ,\mathcal{A}(\alpha_s)A(α1​),⋯,A(αs​) 也线性相关。
  • 反之一般不成立（理解：线性映射是秩不升的）

基底的像确定线性映射：若 α1,⋯ ,αn\alpha_1, \cdots ,\alpha_nα1​,⋯,αn​ 为 KnK^nKn 的一组基，在 KmK^mKm 中指定任意 nnn 个向量 β1,⋯ ,βn\beta_1, \cdots ,\beta_nβ1​,⋯,βn​，则存在唯一的线性映射 A:Kn→Km\mathcal{A}:K^n\to K^mA:Kn→Km s. t. ∀1≤i≤n\forall 1\le i\le n∀1≤i≤n 有 A(αi)=βi\mathcal{A}(\alpha_i) = \beta_iA(αi​)=βi​。

如何理解？考虑 KnK^nKn 中每一个元素都可以写成 k1α1+k2α2+⋯+knαnk_1\alpha_1+k_2\alpha_2+ \cdots +k_n\alpha_nk1​α1​+k2​α2​+⋯+kn​αn​ 的形式，我们定义线性映射

A:Kn⟶Kmk1α1+⋯+knαn⟼k1β1+⋯+knβn\begin{aligned} \mathcal{A}: K^n&\longrightarrow K^m\\ k_1\alpha_1+ \cdots +k_n\alpha_n &\longmapsto k_1\beta_1+ \cdots +k_n\beta_n\\ \end{aligned} A:Knk1​α1​+⋯+kn​αn​​⟶Km⟼k1​β1​+⋯+kn​βn​​

其中 k1,⋯kn∈Kk_1, \cdots k_n\in Kk1​,⋯kn​∈K。容易证明其为线性映射。存在性证毕。

唯一性也是显然的。根据我们如上的定义，对于 ∀α∈Kn\forall \alpha\in K^n∀α∈Kn，其都能被唯一写成 k1α1+⋯+knαnk_1\alpha_1+ \cdots +k_n\alpha_nk1​α1​+⋯+kn​αn​ 的形式，由线性映射的性质，A\mathcal{A}A 若存在则一定唯一。

两个子空间

  • 像空间：不难验证 Im⁡A\operatorname{Im} \mathcal{A}ImA 为 KmK^mKm 的子空间。称为 A\mathcal{A}A 的像空间。
  • 核空间：Ker⁡A:={α∈Kn∣Aα=0}\operatorname{Ker} \mathcal{A} := \left\{ \alpha\in K^n \mid \mathcal{A} \alpha = 0 \right\}KerA:={α∈Kn∣Aα=0} 为 KnK^nKn 的子空间（同样不难验证），称为 A\mathcal{A}A 的核空间。

两个基本事实。设 A∈Mm×n(K)A\in M_{m\times n}(K)A∈Mm×n​(K)，令

A:Kn⟶Kmα⟼Aα\begin{aligned} \mathcal{A}:K^n&\longrightarrow K^m \\ \alpha&\longmapsto A\alpha \end{aligned} A:Knα​⟶Km⟼Aα​

不难发现 A\mathcal{A}A 为线性映射。

  • 设 AAA 的列向量组为 α1,⋯ ,αn\alpha_1, \cdots ,\alpha_nα1​,⋯,αn​，则 Im⁡A=C(A)\operatorname{Im} \mathcal{A} = \mathcal{C}(A)ImA=C(A)。
  • Ker⁡A=N(A)\operatorname{Ker} \mathcal{A} = \mathcal{N}(A)KerA=N(A)。

所以 dim⁡(Ker⁡A)+dim⁡(Im⁡A)=n\operatorname{dim}(\operatorname{Ker} \mathcal{A}) + \operatorname{dim}(\operatorname{Im} \mathcal{A}) = ndim(KerA)+dim(ImA)=n。
