再生核希尔伯特空间（RKHS）

在课上听这个的时候有点懵，这里进行重新整理。

Hilbert 空间

一个希尔伯特空间（Hilbert Space）是对欧氏空间的一个扩展，其还是向量空间，有内积 ⟨f,g⟩H\langle f,g\rangle_{\mathcal{H}}⟨f,g⟩H​。由内积可以定义范数 ∥f∥:=⟨f,f⟩H\left\| f \right\| := \sqrt{\langle f,f\rangle_{\mathcal{H}}}∥f∥:=⟨f,f⟩H​​，且完备（no holes，每个柯西列都收敛到某个点）。

内积要满足的性质：

  • 对称性：⟨f,g⟩H=⟨g,f⟩H\langle f,g\rangle_{\mathcal{H}}=\langle g,f\rangle_{\mathcal{H}}⟨f,g⟩H​=⟨g,f⟩H​；
  • 线性性：⟨af1+bf2,g⟩H=a⟨f1,g⟩H+b⟨f2,g⟩H\langle af_1+bf_2,g\rangle_{\mathcal{H}} = a\langle f_1,g\rangle_{\mathcal{H}} + b\langle f_2,g\rangle_{\mathcal{H}}⟨af1​+bf2​,g⟩H​=a⟨f1​,g⟩H​+b⟨f2​,g⟩H​；
  • 正定性：⟨f,f⟩H≥0\langle f,f\rangle_{\mathcal{H}}\ge 0⟨f,f⟩H​≥0 且 ⟨f,f⟩H=0  ⟺  f=0\langle f,f\rangle_{\mathcal{H}}=0 \iff f = 0⟨f,f⟩H​=0⟺f=0

事实上，定义在集合 X\mathcal{X}X 上的全体函数 f:X→Rf: \mathcal{X}\to \mathbb{R}f:X→R 便可以构成一个希尔伯特空间（函数可以看作为无穷维度的向量，回忆高代书上的内容（x））

RKHS

定义（from Wikipedia）：考虑一个希尔伯特空间，其元素为定义在 X\mathcal{X}X 上的函数，若 ∀x∈X\forall x \in \mathcal{X}∀x∈X，存在函数 Kx∈HK_x \in \mathcal{H}Kx​∈H 使得 ∀f∈H\forall f \in \mathcal{H}∀f∈H 有 f(x)=⟨f,Kx⟩Hf(x) = \langle f, K_x \rangle_{\mathcal{H}}f(x)=⟨f,Kx​⟩H​，则 H\mathcal{H}H 为再生核希尔伯特空间（RKHS）

有点抽象，我们换个角度重新理解，因为理论上这个所谓的 RKHS 就是由核“再生”出来的，所以我们自然从核函数考虑起会自然一些。

考虑一个合法的核函数 K(x,y)K(x,y)K(x,y)，由于我们可以将函数看成无穷维向量，此二元函数自然就可以看成无穷维矩阵。其正定性可以被解读为 ∫∫f(x)K(x,y)f(y)dxdy≥0\displaystyle \int\int f(x)K(x,y)f(y) \mathrm{d}x\mathrm{d}y\ge 0∫∫f(x)K(x,y)f(y)dxdy≥0，对称性可解读为 K(x,y)=K(y,x)K(x,y) = K(y,x)K(x,y)=K(y,x)。

那么同样地，类似对普通矩阵特征值分解，我们也可以对核函数进行类似操作，即存在特征值 λ\lambdaλ 与特征函数 ψ(x)\psi(x)ψ(x) 使得

∫K(x,y)ψ(x)dx=λψ(x)\int K(x,y) \psi(x) \mathrm{d}x = \lambda \psi(x) ∫K(x,y)ψ(x)dx=λψ(x)

特征向量自然是正交的（证明略过），即

⟨ψ1,ψ2⟩H=∫ψ1(x)ψ2(x)dx=0\langle \psi_1,\psi_2 \rangle_{\mathcal{H}} = \int \psi_1(x)\psi_2(x)\mathrm{d}x = 0 ⟨ψ1​,ψ2​⟩H​=∫ψ1​(x)ψ2​(x)dx=0

这说明一个核函数对应着无穷个特征值与无穷个特征方程，与矩阵类似地对其特征值分解：

K(x,y)=∑i=1∞λiψi(x)ψi(y)K(x,y) = \sum_{i=1}^{\infty} \lambda_i \psi_i(x) \psi_i(y) K(x,y)=i=1∑∞​λi​ψi​(x)ψi​(y)

这就是 SVM 一文中所说的 Mercer 定理，这说明所有的 ψi\psi_iψi​ 构成了函数空间的一组正交基。

{λiψi}i=1∞\{\sqrt{\lambda_i}\psi_i\}_{i=1}^{\infty}{λi​​ψi​}i=1∞​ 自然为空间中的一组正交基。用这组基可以张成希尔伯特空间 H\mathcal{H}H，该空间的任意一个函数都可以表示为这组基的线性组合：

f=∑i=1∞fiλiψif = \sum_{i=1}^{\infty} f_i \sqrt{\lambda_i} \psi_i f=i=1∑∞​fi​λi​​ψi​

⟨f,g⟩H=∑i=1∞figi\langle f,g \rangle_{\mathcal{H}} = \sum_{i=1}^{\infty} f_i g_i ⟨f,g⟩H​=i=1∑∞​fi​gi​

对于核函数 KKK，固定一个变量后的 K(x,⋅)K(x,\cdot )K(x,⋅) 为一元函数，用“特征值分解”的等式拆开：

同理，K(⋅,y)K(\cdot ,y)K(⋅,y) 可以表示为 (λ1ψ1(y),λ2ψ2(y),⋯ )(\sqrt{\lambda_1}\psi_1(y), \sqrt{\lambda_2}\psi_2(y), \cdots )(λ1​​ψ1​(y),λ2​​ψ2​(y),⋯)，所以，这二者的内积

先将 fff 分解为两部分：一部分在 $\langle k(x_1,\cdot ),k(x_2,\cdot ), \cdots k(x_n,\cdot ) \rangle $ 上（此处记号为 span），另一部分 vvv 与前一部分正交。于是 ∀f\forall f∀f 可以写成 f=∑i∈[n]αik(xi,⋅)+v\displaystyle f = \sum_{i \in [n]} \alpha_i k(x_i,\cdot ) + vf=i∈[n]∑​αi​k(xi​,⋅)+v，满足 $\langle v,k(x_i,\cdot ) \rangle_{\mathcal{H}} = 0 $。接下来证明 vvv 一定为 000 即可。

考虑一个训练样本点 xjx_jxj​，有

f(xj)=⟨∑i∈[n]αik(xi,⋅)+v,k(⋅,xj)⟩H=∑i∈[n]αi⟨k(xi,⋅),k(⋅,xj)⟩H+∑i∈[n]⟨v,k(⋅,xj)⟩H=∑i∈[n]αik(xi,xj)\begin{aligned} f(x_j) &= \left\langle \sum_{i \in [n]}\alpha_ik(x_i,\cdot ) + v, k(\cdot ,x_j) \right\rangle_{\mathcal{H}}\\ &= \sum_{i \in [n]}\alpha_i \langle k(x_i,\cdot ), k(\cdot ,x_j) \rangle_{\mathcal{H}} + \sum_{i \in [n]}\langle v, k(\cdot ,x_j) \rangle_{\mathcal{H}}\\ &= \sum_{i \in [n]} \alpha_i k(x_i, x_j) \end{aligned} f(xj​)​=⟨i∈[n]∑​αi​k(xi​,⋅)+v,k(⋅,xj​)⟩H​=i∈[n]∑​αi​⟨k(xi​,⋅),k(⋅,xj​)⟩H​+i∈[n]∑​⟨v,k(⋅,xj​)⟩H​=i∈[n]∑​αi​k(xi​,xj​)​

这进而说明损失函数是跟 vvv 无关的。

考虑 R(∥f∥)=R(∥f0+v∥)R(\left\| f \right\|) = R(\left\| f_0+v \right\|)R(∥f∥)=R(∥f0​+v∥)。∥f0+v∥2=⟨f0,f0⟩H+⟨v,v⟩H+2⟨f0,v⟩H≥∥f0∥2\left\| f_0+v \right\|^{2} = \langle f_0,f_0 \rangle_{\mathcal{H}} + \langle v,v \rangle_{\mathcal{H}} + 2\langle f_0,v\rangle_{\mathcal{H}} \ge \left\| f_0 \right\|^{2}∥f0​+v∥2=⟨f0​,f0​⟩H​+⟨v,v⟩H​+2⟨f0​,v⟩H​≥∥f0​∥2。而且等于号成立当且仅当 v=0v=0v=0。又 RRR 严格递增，所以为了使得整体的值最小（损失函数项与 vvv 无关），我们自然是需要 v=0v=0v=0 的。□\square□

直觉上理解起来应该挺自然的，模型的输出结果必然是与训练数据有关的，而正则化项把无关项去掉了。

优势

  • 将一个对 fff 的潜在的无穷维优化变成只对 nnn 个变量的搜索。
    考虑 RBF 核，φ(x)\varphi(x)φ(x) 是无穷维的，wTw^TwT 也应无穷维，这样就无法优化了，但利用表示定理我们就可以对 nnn 个 αi\alpha_iαi​ 进行优化了。
