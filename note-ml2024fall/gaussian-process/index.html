随机过程（Stochastic Process）

定义：A collection of (infinitely) random variables along an index set.

Index set 可以为 N,R,Rd\mathbb{N},\mathbb{R},\mathbb{R}^dN,R,Rd。其可以在描述时间，但更一般性的描述下其也不一定是时间。

考虑 index xix_ixi​，xix_ixi​ 对应的随机变量采样得到 yiy_iyi​。当有无限多组 {(xi,yi)}\{(x_i,y_i)\}{(xi​,yi​)} 时候可以确定一个函数 y(x)y(x)y(x) 的分布，每次采样会得到一个确定的函数 y(x)y(x)y(x)。

为了确定一个随机过程，我们只需要确定其包含的所有随机变量的联合分布（joint distribution）注意，这些随机变量大概率是不独立的。

例 2：Markov 随机过程

{y1,y2,⋯ ,yT}\{y_1,y_2, \cdots ,y_T\}{y1​,y2​,⋯,yT​}，一般认为建立在时间序列上。P(yt+1∣y1,⋯ ,yt)=P(yt+1∣yt)P(y_{t+1}\mid y_1, \cdots ,y_t) = P(y_{t+1}\mid y_t)P(yt+1​∣y1​,⋯,yt​)=P(yt+1​∣yt​)。

高斯过程（Gaussian Process）

定义与求解推导

joint distribution over any finite collection of R.V.s to be Gaussian.

考虑任意 ∀{x1,x2,⋯ ,xn}⊆X\forall \{x_1,x_2, \cdots ,x_n \}\subseteq X∀{x1​,x2​,⋯,xn​}⊆X，其中 XXX 为标号集合，则其对应的随机变量 {y1,y2,⋯ ,yn}\{y_1,y_2, \cdots ,y_n\}{y1​,y2​,⋯,yn​} 为高斯过程当且仅当 y1,⋯ ,yny_1, \cdots ,y_ny1​,⋯,yn​ 服从高斯分布。

为了确定一个高斯过程，我们可以对于每个随机变量 yiy_iyi​，确定其 μi\mu_iμi​，以及对于每个 i,ji,ji,j 对，确定 Σij\Sigma_{ij}Σij​。但是如果随机变量的个数很多，而且再考虑往里面添加新的随机变量的话，需要确定的参数数量就会呈指数级爆炸。所以我们希望有一种 consistent 的方式来进行求解。

考虑求出两个函数 m(⋅),k(⋅,⋅)m(\cdot ),k(\cdot,\cdot )m(⋅),k(⋅,⋅)，其以随机变量 yiy_iyi​ 对应的下标 xix_ixi​ 作为输入（不能以随机变量作为输入，毕竟我们就想要的是均值和协方差），分别输出 μi\mu_iμi​ 和 Σij\Sigma_{ij}Σij​。

其实这个 k(⋅,⋅)k(\cdot ,\cdot )k(⋅,⋅) 就是核函数，考虑到核函数的含义是给出两个向量的相似度，在这里就可以变相地说明 yiy_iyi​ 和 yjy_jyj​ 的相似程度——下标 xi,xjx_i,x_jxi​,xj​ 越接近，yi,yjy_i,y_jyi​,yj​ 越相关，这也是很符合直觉的（geometric meaning）。

有了这两个函数之后，我们就可以将一个高斯过程写成

y(x)∼GP⁡(m(x),k(x,x′))y(x) \sim \operatorname{GP}(m(x), k(x,x')) y(x)∼GP(m(x),k(x,x′))

一般而言，我们都直接令 m(x)=0m(x) = 0m(x)=0（回忆贝叶斯角度下的线性回归），在无关于 yyy 的先验知识下令均值为 000 是很符合 Occam’s Razar 原理的。

同时，Σ\SigmaΣ 应该是正定的。合法的核函数已经满足其半正定性。如果出现了有特征值为 000 的情况，可以通过添加 λI\lambda IλI 噪声来进行解决。

考虑求解 kkk。我们其实一般使用 RBF kernel：

k(xi,xj)=exp⁡(−∥xi−xj∥22l2)k(x_i,x_j) = \exp\left( - \frac{\left\| x_i - x_j \right\|^{2}}{2 l^2} \right) k(xi​,xj​)=exp(−2l2∥xi​−xj​∥2​)

令 KKK 为 nnn 个训练点 x1,⋯ ,xnx_1, \cdots ,x_nx1​,⋯,xn​ 对应的 Gram 矩阵，其中 Kij=k(xi,xj)K_{ij} = k(x_i,x_j)Kij​=k(xi​,xj​)，K∈Rn×nK \in R^{n\times n}K∈Rn×n，然后令 y:=[y1,⋯ ,yn]Ty := [y_1, \cdots ,y_n]^Ty:=[y1​,⋯,yn​]T 为 nnn 维随机变量。

则我们有 P(y)=N(y∣0,K)P(y) = \mathcal{N}(y\mid 0,K)P(y)=N(y∣0,K)，其为训练数据的标签的联合分布。

但是光有这个没有用，对于训练数据，我们是知道 GT 的，即知道每个 yiy_iyi​ 分别是什么，我们更需要的是给定一个新的测试点 x∗x^*x∗，求出 P(y∗∣y)P(y^*\mid y)P(y∗∣y)。

P(y∗∣y)=N(y∗∣μ∗,Σ∗)P(y^*\mid y) = \mathcal{N}(y^*\mid \mu^*, \Sigma^*) P(y∗∣y)=N(y∗∣μ∗,Σ∗)

Standard Conclusions from Gaussian

对于一个随机变量

  1. 边缘化（marginalization）：

    xa∼N(μa,Σaa)xb∼N(μb,Σbb)\begin{aligned} x_a &\sim \mathcal{N}(\mu_a, \Sigma_{aa}) \\ x_b &\sim \mathcal{N}(\mu_b, \Sigma_{bb}) \end{aligned} xa​xb​​∼N(μa​,Σaa​)∼N(μb​,Σbb​)​

    P(xa∣xb)=N(xa∣μa∣b,Σa∣b) P(x_a\mid x_b) = \mathcal{N}(x_a\mid \mu_{a\mid b}, \Sigma_{a\mid b}) P(xa​∣xb​)=N(xa​∣μa∣b​,Σa∣b​)

    其中

    μa∣b=μa+ΣabΣbb−1(xb−μb)Σa∣b=Σaa−ΣabΣbb−1Σba \begin{aligned} \mu_{a\mid b} &= \mu_a + \Sigma_{ab}\Sigma_{bb}^{-1}(x_b - \mu_b) \\ \Sigma_{a\mid b} &= \Sigma_{aa} - \Sigma_{ab}\Sigma_{bb}^{-1}\Sigma_{ba} \end{aligned} μa∣b​Σa∣b​​=μa​+Σab​Σbb−1​(xb​−μb​)=Σaa​−Σab​Σbb−1​Σba​​

    这个式子会在之后的部分反复用到。

补充：张老师关于该式子物理含义的理解

考虑 μa∣b\mu_{a\mid b}μa∣b​：该式子在描述在标准化意义下，xbx_bxb​ 偏离其均值有多大。除以 Σbb\Sigma_{bb}Σbb​ 相当于在做标准化，而 Σab\Sigma_{ab}Σab​ 的含义为 xax_axa​ 与 xbx_bxb​ 倾向于一起怎么样变动。xbx_bxb​ 偏离均值越大，相应地对 μa\mu_aμa​ 影响也会越大。

对于 Σab\Sigma_{ab}Σab​：确定了 xbx_bxb​ 之后，xax_axa​ 的“不确定度”就会减小，所以 Σaa\Sigma_{aa}Σaa​ 后面跟着的是减号。减小的程度与 xa,xbx_a,x_bxa​,xb​ 的相关程度也是成正比的——越相关，确定 xbx_bxb​ 后就“越能确定” xax_axa​，但还是会反比于 xbx_bxb​ 的“不确定度”——若 xbx_bxb​ 本身不确定程度就很大，那么观察到后对 xax_axa​ 的影响也就应该没那么大。

对其有直觉上的理解可以方便记忆。

贝叶斯优化（Bayesian Optimization, BO）

先介绍黑箱优化（black-box optimization），其目标函数是没有解析形式的，更不用提梯度了。传统的梯度下降法对于黑箱优化是无效的。

我们只能做的是，不断喂进去 xxx，然后观测其输出的 yyy。现在的目标就是在尽可能少的次数下使得 yyy 最大化。神经网络的超参数调参就是一种典型的黑箱优化问题。

误区：可能会觉得神经网络不是可以用梯度下降优化吗？

注意我们这里讨论的是超参数，神经网络最后输出的 performance 是无法被写成超参数的解析形式的。

贝叶斯优化的流程如下：

  1. 随机采样若干点，并扔进黑箱求值，得到初始的 {(xi,yi)}\{(x_i,y_i)\}{(xi​,yi​)}；
  2. 用这些点做一个 GPR；
  3. 在回归结果的基础上，利用采集函数（acquisition function）a(x)a(x)a(x) 来选择新的一批点，并将这些点加入已知训练点，重新做 GPR；
  4. 重复第三步直到满足某种停止条件。

采集函数扮演着非常重要的角色，常见的有如下几种：

  • Expected Improvement (EI)：a(x)=E[(y(x)−ymax⁡)+]a(x) = \mathbb{E}[(y(x) - y_{\max})^+]a(x)=E[(y(x)−ymax​)+]，寻找期望意义下能给 ymax⁡y_{\max}ymax​ 带来最大改进的点。
  • Upper Confidence Bound：a(x)=μ(x)+κσ(x)a(x) = \mu(x) + \kappa \sigma(x)a(x)=μ(x)+κσ(x)。和 MCTS 里面的那个很像，相当于在平衡利用（exploitation）和探索（exploration）。
